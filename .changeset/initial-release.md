Initial alpha release. Tenderlane is a reactive, frontend-first payment orchestration SDK. Ships with rules-based PSP routing, headless checkout state machine, React bindings, provider-agnostic catalog primitive, and adapters for Stripe Checkout (redirect) and Polar.sh (redirect).
